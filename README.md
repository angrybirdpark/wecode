>오늘 시험에서 사용했던 git 명령어

1. git init : git 초기화
2. git checkout -b 브랜치명
    * checkout : 브랜치 변경
    * -b : 브랜치 생성을 동시에 진행
3. git add : 로컬 git에서 변경된 내용을 저장한다.
4. git commit -m 메세지
    * add로 스테이징되었던 변경사항을 확정한다.
    * -m 메세지 : 깃허브에 공유될 변경사항 내용을 간략하게 메세지로 입력
5. git push : 로컬 git에 commit된 내용을 github에 보내준다.
